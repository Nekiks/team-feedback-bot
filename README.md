**ПРОЕКТ НЕ ПОДДЕРЖИВАЕТСЯ С МАЯ 2025 ГОДА**

Telegram-бот для сбора обратной связи (обращений) внутри команд. Пользователи объединяются в команды по пригласительным ссылкам, отправляют обращения (в том числе анонимные), а лидеры команд и администраторы обрабатывают их. Бот построен на **aiogram 3** с локальным хранением данных в **SQLite** и экспортом баз данных в JSON для администратора. **Бот работает полностью на стороне сервера и не требует подключения внешних устройств - нужен только токен Telegram-бота.**

---

## Основные возможности

* **Регистрация и роли пользователей:** Автоматическая регистрация при первом `/start`, поддержка ролей участника, лидера команды и администратора. Меню динамически меняется в зависимости от прав пользователя.
* **Управление командами:** Создание команд, генерация пригласительных ссылок, вступление по ссылке, выход из команды, просмотр участников и удаление команды.
* **Обратная связь (обращения):** Отправка обращений с выбором категории, поддержка **анонимных** и **именных** обращений, уведомление лидера команды о новом обращении, обработка и закрытие обращений.
* **Локальное хранилище SQLite:** Данные хранятся в трёх базах в директории `database/` — `users.db`, `teams.db`, `feedbacks.db`.
* **Экспорт баз для администратора:** Администратор может выгрузить любую из баз данных прямо в чат в виде JSON-файла (генерируется через `pandas` в директорию `databases_json/`).
* **Авто-инициализация БД:** При запуске бот сам создаёт необходимые таблицы (`users`, `teams`, `feedbacks`), если их ещё нет.

## Быстрый старт

### 1. Подготовка окружения
Рекомендуется использовать виртуальное окружение:
```bash
python3 -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
```

Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

### 2. Настройка токена
Получите токен у [@BotFather](https://t.me/BotFather) и впишите его в `configs/config_bot.py`:
```python
BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА'
```


### 3. Запуск
```bash
python3 bot_main.py
```
При первом запуске бот автоматически создаст таблицы в базах данных `database/users.db`, `database/teams.db` и `database/feedbacks.db`, после чего начнёт принимать сообщения.

## Использование

1. Откройте бота в Telegram и отправьте команду `/start` — вы будете автоматически зарегистрированы.
2. Команда `/menu` открывает главное меню. Доступные пункты зависят от вашей роли:
   * **Участник:** создание обращения, просмотр своей команды и профиля.
   * **Лидер команды:** дополнительно — управление обращениями своей команды.
   * **Администратор:** дополнительно — панель администратора с возможностью выгрузки баз данных.
3. Чтобы пригласить участников, лидер создаёт команду и генерирует пригласительную ссылку (deep-link вида `https://t.me/<bot>?start=invite...`).

## Базы данных

Проект использует три локальные базы SQLite, создаваемые автоматически:

| База            | Таблица     | Назначение                                              |
|-----------------|-------------|---------------------------------------------------------|
| `users.db`      | `users`     | Пользователи, их роли, команда и статистика обращений   |
| `teams.db`      | `teams`     | Команды, лидеры, пригласительные ссылки и счётчики       |
| `feedbacks.db`  | `feedbacks` | Обращения: автор, команда, категория, статус, анонимность |
